# Tetris
Fun & Simple Original tetris

## Download
[Tetris.jar](https://github.com/salifm/Tetris/raw/master/out/Tetris.jar)


## How to play

* `up` & `down` arrow keys: rotate a tetromino
* `right` & `left` arrow keys: move a tetromino
* `p` key: pause game / restart game
